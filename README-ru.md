# Эмулятор EA App
**[简体中文](README.md) | [English](README-en.md) | Русский**

Эмулятор EA App за авторством `Misaka_Mikoto_01`, `CrazyZhang666`, и `B1397KB`

> Данный проект разрабатывается в качестве простой замены EA App/Origin. Тем более, новый EA App попросту ужасен.

## Скачать

https://github.com/CrazyZhang666/EAappEmulater/releases

## Переключение языка
Программа поддерживает переключение языка. Вы можете выбрать необходимый вам язык в настройках программы.

<img width="1496" height="968" alt="image" src="https://github.com/user-attachments/assets/193be31c-41e1-43ff-af22-c251627d97b0" />

Если вы хотите перевести программу на свой язык, для этого вам необходимо сделать форк данного репозитория и создать языковой файл в директории `Assets/Files/Lang`, следуя примеру `zh-CN.xaml`.

По умолчанию программа попытается прочитать язык интерфейса вашей системы и установит его в качестве такового в программе.

## Функции

- Идеальная поддержка всей серии игр Battlefield, а также других игр от EA. Запуск игр происходит без участия EA App.
- Запуск игры через нативный метод без необходимости модифицировать файлы
- Идеальная совместимость с Frosty
- Идеальная совместимость с играми, использующими анти-чит EAAC
- Поддержка нескольких учётных записей с возможностью переключения между ними (поддержка до 100 штук)
- Расширенный функционал записи логов
- Персонализация настроек запуска игры (поддерживаются настройки из реестра и собственные)
- Смена языка игры
- Запуск игр напрямую, минуя Steam
- Возможность обойти ограничения по языку игры, наложенные на учётную запись игрока
- Встроенный интерфейс EA App для дополнения некоторых функций эмулятора (покупка игр, добавление/удаление друзей, чат с друзьями и т.д.)

## Недостатки

- Нельзя скачать игры и проверить наличие обновлений
- Нельзя использовать облачные сохранения
